== Installation ==
Requires: Woocommerce 2.x, 3.x

= Automatic installation =
To do an automatic install of Triggmine, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Triggmine” and click Search Plugins. Once you’ve found the plugin you can install it by simply clicking “Install Now”.

= Manual installation =
The manual installation method involves downloading the plugin and uploading it to your webserver via FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Configure the plugin =
After the installation activate the plugin in Wordpress Plugins menu. Navigate to the plugin settings using the Triggmine link in the left panel.
1. Enable the plugin choosing \"Yes\" in the corresponding setting.
2. Copy API URL and API key from your Triggmine account and paste them to the corresponding fields of plugin configuration in Wordpress.
3. Save the plugin settings.
